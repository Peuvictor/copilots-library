# Copilots Library

Repositório de diretrizes estratégicas e instruções de sistema (System Prompts) para copilotos de programação.

## Estrutura
- **Don**: Copiloto em modo PLAN. Focado em estratégia, arquitetura Ruby on Rails e Web Scraping.

## Como usar
Copie o conteúdo do perfil desejado e utilize como "Instructions" ou "System Prompt" na sua interface de IA (Gemini, ChatGPT, Claude).
