# 🧠 My Copilots Library

Repositório central de instruções de sistema (System Prompts) personalizadas para o ecossistema **Ruby on Rails**. Esta biblioteca padroniza a inteligência das IAs para garantir rigor técnico, estratégia e execução sólida.

## 🕴️ O Conselho de Especialistas

Cada perfil abaixo possui uma missão específica no ciclo de desenvolvimento, permitindo que eu alterne entre planejamento, execução, diagnóstico e aprendizado de forma organizada.

### 🕵️‍♂️ [Don (Modo PLAN)](./profiles/plan-don/ruby-rails.md)
**O Estrategista.** Don é responsável por olhar o quadro completo. Ele não escreve código antes de produzir um plano revisável.
- **Quando usar:** Ao iniciar novas features, refatorações complexas ou mudanças de arquitetura.
- **Foco:** Riscos, trade-offs, arquivos afetados e passos incrementais para evitar a paralisia por análise (TDAH-friendly).

### 🏗️ [Atlas (Modo AGENT CODE)](./profiles/agent-atlas/ruby-rails.md)
**O Engenheiro.** Atlas é focado na solidez da construção. Ele transforma o plano do Don em código real, limpo e testado.
- **Quando usar:** Para escrever o código propriamente dito, criar services, controllers e migrations.
- **Foco:** Implementação de alta qualidade, cobertura de testes (RSpec) e segurança pela base.

### 🔍 [Mira (Modo ASK)](./profiles/ask-mira/ruby-rails.md)
**A Analista.** Mira é sua lente de diagnóstico. Ela é um agente de "somente leitura" que traz clareza para o que já existe.
- **Quando usar:** Para explicar trechos de código legados, diagnosticar bugs através de logs ou debater alternativas rápidas.
- **Foco:** Clareza, objetividade e diagnóstico preciso de erros (Stack Traces).

### 👩‍🏫 [Dora (Modo STUDY)](./profiles/study-dora/ruby-rails.md)
**A Mentora.** Dora é responsável pelo aprendizado contínuo. Ela não resolve o problema rápido, ela garante que você aprenda o conceito.
- **Quando usar:** Ao encontrar uma gem nova, um conceito de POO complexo ou para se preparar para entrevistas técnicas.
- **Foco:** Didática, analogias, fundamentos de engenharia e correção criteriosa.

---

## 🚀 Como Utilizar

1. Escolha o especialista adequado para sua fase atual no projeto.
2. Copie o conteúdo do arquivo `.md` correspondente.
3. Cole nas **Instruções de Sistema** (System Instructions) ou no início do chat com sua IA.
4. Para melhores resultados, mantenha cada agente em seu respectivo "papel".
